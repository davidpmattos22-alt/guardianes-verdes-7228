# Guardianes Verdes 7228 - Página Web

Esta es una página web sobre educación ambiental, ética y tecnología.

## Estructura de archivos:
- index.html: Página principal con el artículo y sección de comentarios
- css/style.css: Estilos para la página
- js/script.js: Funcionalidad para los comentarios (almacenamiento local)
- images/: Carpeta para imágenes (debes agregar las imágenes necesarias)

## Características:
- Diseño responsivo
- Sistema de comentarios con almacenamiento local
- Secciones organizadas con información sobre medio ambiente
- Código de colores para los tachos ecológicos

## Cómo usar:
1. Coloca todos los archivos en tu servidor web
2. Agrega imágenes relevantes en la carpeta "images"
3. Personaliza los colores y textos según necesites

Los comentarios se guardan en el almacenamiento local del navegador.
